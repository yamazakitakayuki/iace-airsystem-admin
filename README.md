## 開発コマンド

```bash
# ビルド
cd /home/user/admin-webapp
npm run build

# 開発サーバー起動（PM2）
pm2 start ecosystem.config.cjs

# ポートクリーンアップ
npm run clean-port

# サービステスト
npm run test

# PM2管理
pm2 list
pm2 logs admin-webapp --nostream
pm2 restart admin-webapp
pm2 delete admin-webapp
```
